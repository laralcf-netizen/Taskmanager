# 📋 Gerenciador de Tarefas

Aplicação de terminal desenvolvida em Java para gerenciar tarefas do dia a dia, com menu interativo e tratamento de exceções customizadas.

## 💡 Sobre o projeto

src/

├── app/  → Main.java (ponto de entrada)

├── controller/   → TaskManager.java (lógica de negócio)

├── exceptions/   → TarefaException.java (exceção customizada)

└── model/        → Tarefa.java (modelo de dados)

Este projeto foi desenvolvido como parte dos meus estudos de **Programação Orientada a Objetos (POO)** em Java, aplicando conceitos como herança, encapsulamento, exceções customizadas e organização em pacotes.

## 🚀 Funcionalidades

- ✅ Criar nova tarefa (título e descrição)
- 📄 Listar todas as tarefas cadastradas
- ✔️ Concluir tarefa por ID
- 🗑️ Remover tarefa por ID
- 🔁 Menu interativo no terminal
